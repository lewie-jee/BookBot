# BookBot

BookBot is a command-line application built as part of the Boot.dev Python curriculum.

It analyzes a text file and generates useful statistics about its contents, including word counts and character frequency analysis.

This was my first Python project and introduced me to file processing, data structures, sorting algorithms, and command-line applications.

---

## 🚀 Features

- Counts the total number of words in a text file
- Calculates character frequencies
- Sorts and displays character statistics
- Accepts a file path through command-line arguments
- Generates a simple report directly in the terminal

## 📦 Installation

### Prerequisites

- Python 3.10+ (or newer)

### Clone the Repository

```bash
git clone <repository-url>
cd bookbot
```

---

## ▶️ Running the Program

Analyze the included copy of *Frankenstein*:

```bash
python3 main.py books/frankenstein.txt
```

Or analyze your own text file:

```bash
python3 main.py path/to/book.txt
```
