# bookbot

BookBot is my first [Boot.dev](https://www.boot.dev) project!

A Python command-line application that analyzes a text file and generates statistics about its contents.

## Features

- Counts total words in a text
- Calculates character frequencies
- Sorts and displays character statistics
- Accepts a file path through command-line arguments
